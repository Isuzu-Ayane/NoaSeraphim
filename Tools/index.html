Noaplugin (Browser Extension)

【 エロアニメショコラ、favhentaiのダウンローダーはこちら 】

このプラグインは、ChromeおよびEdgeブラウザ専用の拡張機能です。
対象サイトからの動画ダウンロードをサポートします。

🚀 インストール手順

  1. 下のボタンから Noaplugin.zip をダウンロードし、フォルダを展開（解凍）します。
  2. ブラウザのアドレスバーに chrome://extensions/ (または edge://extensions/) と入力して拡張機能ページを開きます。
  3. 右上の 「デベロッパー モード」 をオンにします。
  4. 左上の 「パッケージ化されていない拡張機能を読み込む」 をクリックします。
  5. 解凍した中に含まれる Noaplugin フォルダを選択して読み込ませます。
⬇ Noaplugin.zip をダウンロード

AI適正スペックツール（Python）

あなたのPC環境をスキャンし、ローカルAI（画像生成やLLMなど）に適しているかを診断する専用スクリプトです。
ターミナルでお使いのPCの構成やVRAM、CUDAの利用可能状況が一目でわかります。

⚠️ 実行時の注意事項

  • このツールは Pythonスクリプト (.py) です。Windows環境で実行するには、Python本体のインストールが必要です。
  • ダウンロード後、コマンドプロンプトまたはPowerShellから python system_info.py と入力して実行してください。
  • 実行前にターミナルの文字コードがUTF-8であることを推奨します。
